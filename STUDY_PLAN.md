## Readiness Criteria

- You can explain workflow vs. agent without notes.
- You can list the five workflow patterns.
- You can describe the agent loop with `stop_reason`, `tool_use`, and `tool_result`.
- You can name all ten classic anti-patterns.
- You can choose between prompt, hook, tool, MCP server, workflow, agent, and subagent for a scenario.
- You can explain validation-retry for structured output.
- You can explain prompt caching layout.
- You can explain why aggregate metrics can hide slice failures.
- You can finish mixed questions at roughly 2 minutes per question.
