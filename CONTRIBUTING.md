# Contributing

Contributions are welcome if they keep the repository clean, original, and study-only.

## Allowed

- Original explanations.
- Public-docs-based summaries in your own words.
- New concept drills.
- Corrections to version-sensitive API behavior.
- Links to official sources.
- Better organization and study plans.

## Not Allowed

- Real exam questions.
- Proctored exam screenshots or transcripts.
- Paid course material.
- PDF redistribution without explicit permission.
- Large copied passages from Anthropic or third-party docs.
- Claims that cannot be traced to public documentation or clearly labeled experience.
- Anything intended for use during a live, graded, monitored, or proctored exam.

## Style

- Keep explanations concise but complete.
- Prefer decision rules and scenario cues over memorization.
- Mark version-sensitive behavior clearly.
- Link to official docs whenever possible.
- Treat practice questions as concept drills, not exam replicas.

## Pull Request Checklist

- [ ] I wrote the content myself or have permission to publish it.
- [ ] I did not include real exam content.
- [ ] I added sources for factual claims.
- [ ] I avoided long copied passages.
- [ ] I used ASCII punctuation unless a file already requires otherwise.
