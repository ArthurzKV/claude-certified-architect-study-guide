# Notice And Source Boundary

This repository is an unofficial study guide. It is not affiliated with Anthropic.

## Included Content

The included Markdown files are original study notes, summaries, decision frameworks, and practice drills based on publicly accessible documentation and research. They are written for concept learning and are not official exam questions.

## Excluded Content

The public repository intentionally excludes:

- Downloaded PDFs.
- Screenshots.
- OCR/camera captures.
- Runtime logs.
- RAG app code.
- Official exam questions.
- Practice exam screenshots/transcripts.
- Third-party Markdown repositories without a clear redistribution license.

## Public Sources To Verify Against

Use these as primary references:

- Anthropic Partner Network announcement: https://www.anthropic.com/news/claude-partner-network
- Anthropic Building Effective Agents: https://www.anthropic.com/research/building-effective-agents

## Takedown / Correction

If any file appears to include material that should not be public, open an issue with the path and reason. The policy is to remove questionable material first, then replace it with a clean original summary.
